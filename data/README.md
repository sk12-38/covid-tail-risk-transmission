# Data

Raw price data is not included in this repository.

The analysis scripts download ETF price data from Yahoo Finance and KRX using `yfinance` and `pykrx`. Generated CSV files should stay local and should not be committed.

Main asset universe:

- `ACWI`, `ACWX`, `EFA`, `VWO`
- `069500`, `229200`
- `BNDX`, `EMB`, `IEF`, `TLT`
- `114260`, `148070`
- `USO`, `GLD`, `VNQ`
